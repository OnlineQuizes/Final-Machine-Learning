1. In supervised learning the input contains?

Correct Answer: b

Explanation: Supervised learning requires both input features and corresponding output labels to train a model that can predict outputs for new inputs. The algorithm learns the mapping from inputs to outputs.

2. Examples of some data without outputs is the input of ‚Ä¶

Correct Answer: d

Explanation: Unsupervised learning works with unlabeled data (data without outputs). It discovers patterns, structures, or relationships in the data without predefined categories.

3. Classification and Regression are types of ‚Ä¶ Learning

Correct Answer: a

Explanation: Both classification (predicting discrete labels) and regression (predicting continuous values) are supervised learning tasks because they require labeled training data.

4. The output in supervised learning is a model that predicts unknown ‚Ä¶ given a new input.

Correct Answer: a

Explanation: The trained supervised learning model takes new inputs and predicts the corresponding outputs based on patterns learned from the training data.

5. Reinforcement Learning is to learn how to perform a task from interactions with the environment.

Correct Answer: a

Explanation: Reinforcement learning involves an agent learning to make decisions by interacting with an environment to maximize cumulative reward through trial and error.

6. An example of applications of ML includes ‚Ä¶

Correct Answer: d

Explanation: Machine learning has diverse applications across medicine (diagnosis), scientific research (discovering patterns), and generative AI (creating content like images and text).

7. In linear regression we find the ‚Ä¶

Correct Answer: b

Explanation: Linear regression finds the line that best fits the data points by minimizing the sum of squared errors between observed and predicted values.

8. In linear regression the aim is to reduce the cost value as much as we can.

Correct Answer: a

Explanation: The cost function (like Mean Squared Error) measures prediction error. Minimizing it through optimization algorithms like gradient descent improves model accuracy.

9. ‚Ä¶ is an example of regression evaluation metrics.

Correct Answer: d

Explanation: All listed metrics (MSE, RMSE, R¬≤) are used to evaluate regression models by measuring prediction accuracy and goodness of fit.

10. MAE calculates the absolute difference between actual and predicted values using ‚Ä¶

Correct Answer: a

Explanation: MAE (Mean Absolute Error) calculates the average absolute difference between predicted and actual values, making it less sensitive to outliers than MSE.

12. The Root Mean Squared Error is calculated as follows:

Correct Answer: a

Explanation: RMSE is the square root of MSE, putting error back in the original units of the target variable for easier interpretation.

13. ‚Ä¶ shows how well the data fit the regression model (goodness of fit).

Correct Answer: c

Explanation: R¬≤ (coefficient of determination) measures how well the model explains variance in the dependent variable, ranging from 0 (poor fit) to 1 (perfect fit).

14. R¬≤-Score is calculated using the equation:

Correct Answer: c

Explanation: R¬≤ = 1 - (SSR/SST), where SSR is sum of squared residuals and SST is total sum of squares. It represents proportion of variance explained.

15. Logistic Regression (LR) is a type of ‚Ä¶

Correct Answer: c

Explanation: Despite its name, logistic regression is used for binary classification (e.g., spam detection; Predict if email is spam (1) or not spam (0)) by estimating probabilities using the sigmoid function.

16. The function used to calculate the prediction model in Logistic Regression is ‚Ä¶

Correct Answer: a

Explanation: The sigmoid function (logistic function) maps any real-valued number to a probability between 0 and 1, making it suitable for binary classification.

17. In Logistic Regression, the line that makes perfect separation between data is called ‚Ä¶

Correct Answer: b

Explanation: The decision boundary is the threshold (typically 0.5 probability) where the model separates predicted classes, derived from the logistic function.

18. We use ‚Ä¶ to minimize the cost function in Logistic Regression.

Correct Answer: c

Explanation: Gradient descent iteratively adjusts model parameters to minimize the logistic loss (cross-entropy) function by moving in the direction of steepest descent.

19. A ‚Ä¶ is a matrix that summarizes the performance of a machine-learning model.

Correct Answer: d

Explanation: A confusion matrix displays true positives, false positives, true negatives, and false negatives, providing a detailed view of classification performance.

20. The most commonly used metric to judge a model is ‚Ä¶

Correct Answer: a

Explanation: Accuracy = (TP+TN)/Total is intuitive but can be misleading with imbalanced classes. It's often supplemented with precision, recall, and F1-score.

21. Clustering is a type of ‚Ä¶ Learning

Correct Answer: c

Explanation: Clustering groups similar data points without predefined labels, making it unsupervised learning. Common algorithms include K-means and hierarchical clustering.

22. Accuracy is calculated as:

Correct Answer: b

Explanation: Accuracy = (True Positives + True Negatives) / Total Predictions. It measures overall correctness but can be skewed with class imbalance.

23. Precision is calculated as:

Correct Answer: a

Explanation: Precision = TP/(TP+FP). Also called positive predictive value, it measures how many predicted positives are actually positive (important when FP cost is high).

24. Recall is calculated as:

Correct Answer: b

Explanation: Recall = TP/(TP+FN). Also called sensitivity or true positive rate, it measures how many actual positives are correctly identified (important when FN cost is high).

25. F1 Score is calculated as:

Correct Answer: c

Explanation: F1 Score = 2 √ó (Precision √ó Recall) / (Precision + Recall). It's the harmonic mean of precision and recall, balancing both metrics.

26. Macro Average is calculated as:

Correct Answer: a

Explanation: Macro-average computes the metric independently for each class and then takes the unweighted mean, treating all classes equally regardless of size.

27. Weighted Average is calculated as:

Correct Answer: c

Explanation: Weighted average calculates metrics for each class and averages them weighted by the number of true instances for each class (class support).

28. A well-defined ML task is described by:

Correct Answer: b

Explanation: Tom Mitchell's definition: A computer program learns from experience E with respect to task T and performance measure P if its performance on T improves with E.

29. The goal of clustering is to ‚Ä¶

Correct Answer: b

Explanation: Clustering discovers natural groupings or patterns in unlabeled data, revealing structure like customer segments, document topics, or image features.

30. Reinforcement learning learns from‚Ä¶

Correct Answer: b

Explanation: RL agents learn through trial and error by taking actions, receiving rewards/penalties, and updating policies to maximize cumulative reward over time.

31. The cost function in linear regression is also known as:

Correct Answer: b

Explanation: Cost function (or loss/error function) quantifies prediction error. Mean Squared Error is commonly used for linear regression.

32. One-vs-All classification converts multi-class into ‚Ä¶

Correct Answer: b

Explanation: One-vs-All trains one binary classifier per class (class vs all others). For prediction, the classifier with highest confidence wins.

33. In One-vs-One classification, number of classifiers needed is ‚Ä¶

Correct Answer: d

Explanation: One-vs-One creates a classifier for each pair of classes. For N classes, you need N(N-1)/2 classifiers (combinations of N taken 2 at a time).

34. MLE refers to a method that:

Correct Answer: b

Explanation: Maximum Likelihood Estimation finds parameter values that maximize the likelihood function, making the observed data most probable under the model.

35. Naive Bayes assumes features are ‚Ä¶

Correct Answer: b

Explanation: The "naive" assumption is that features are conditionally independent given the class label, simplifying probability calculations dramatically.

36. Bayes' theorem computes:

Correct Answer: b

Explanation: Bayes' theorem: P(A|B) = P(B|A)P(A)/P(B). In ML, it computes posterior probability of a class given the features.

37. Naive Bayes chooses the class with:

Correct Answer: c

Explanation: Naive Bayes predicts the class with maximum posterior probability P(class|features), calculated using Bayes' theorem with the independence assumption.

38. A single coin flip follows a ‚Ä¶ distribution

Correct Answer: c

Explanation: Bernoulli distribution models a single binary trial (success/failure) with probability p. Coin flip is the classic example.

39. What is a neural network in machine learning?

Correct Answer: c

Explanation: Neural networks are computational models inspired by biological neural networks, consisting of interconnected nodes (neurons) that process information.

40. The process in Neural Networks that mimics biological neurons is used for:

Correct Answer: d

Explanation: Neural networks perform all these functions: pattern recognition (computer vision), weighted decision making (layers with weights), and prediction (output layer).

41. Which of the following is NOT a characteristic of neural networks?

Correct Answer: c

Explanation: Neural networks use non-linear activation functions to model complex patterns. Pure linear computation would limit their expressive power (perceptron limitation).

42. Which of the following layers is the first in a Neural Network?

Correct Answer: c

Explanation: The input layer receives the raw data/features. Each neuron typically corresponds to one feature in the dataset.

43. What is the primary purpose of the hidden layer in a neural network?

Correct Answer: b

Explanation: Hidden layers transform inputs through weighted connections and non-linear activations to learn hierarchical representations of the data.

44. What is the primary purpose of the output layer in a neural network?

Correct Answer: c

Explanation: The output layer produces the final predictions. For regression: single neuron with linear activation; for classification: multiple neurons with softmax/sigmoid.

45. In Neural Networks, the number of neurons in the output layer depends on:

Correct Answer: c

Explanation: Regression: 1 neuron. Binary classification: 1 neuron with sigmoid. Multi-class: N neurons with softmax (N = number of classes).

46. What is the primary advantage of using multiple hidden layers in a neural network?

Correct Answer: a

Explanation: Deep networks (multiple hidden layers) can learn hierarchical feature representations, capturing complex patterns and improving accuracy for difficult tasks.

47. What does a neuron represent in a neural network?

Correct Answer: c

Explanation: A neuron computes: output = activation(Œ£(weights √ó inputs) + bias). It's the fundamental computational unit of neural networks.

48. What is the main role of bias in a Neural Network?

Correct Answer: b

Explanation: Bias allows shifting the activation function, giving the model flexibility to fit data better. Without bias, the decision boundary would always pass through origin.

49. What is the main role of the weights in a neural network?

Correct Answer: b

Explanation: Weights represent connection strengths between neurons. During training, they're adjusted to minimize error, learning which features are important.

50. What is the primary function of an activation function in a neural network?

Correct Answer: b

Explanation: Without non-linear activations, multiple layers would collapse to a single linear transformation, unable to learn complex patterns (universal approximation theorem).

51. Which of the following is NOT an activation function?

Correct Answer: d

Explanation: Normalization (e.g., batch norm) is a preprocessing/regularization technique, not an activation function. Activation functions transform neuron outputs non-linearly.

52. The function \( f(x) = \frac{1}{1+e^{-x}} \) is known as:

Correct Answer: b

Explanation: Sigmoid (logistic function) outputs values between 0 and 1, useful for probability estimation. Prone to vanishing gradients in deep networks.

53. In a Perceptron, what happens when the predicted output does not match the actual output?

Correct Answer: b

Explanation: Perceptron learning rule: if misclassified, adjust weights: w = w + Œ∑(y - ≈∑)x. Œ∑ is learning rate, y is true label, ≈∑ is prediction.

54. What does a perceptron consist of?

Correct Answer: c

Explanation: A single-layer perceptron has input nodes directly connected to output node(s) with no hidden layers. Limited to linearly separable problems.

55. In which of the following neural networks is the data passed in a forward direction through multiple layers?

Correct Answer: c

Explanation: Feedforward networks pass information forward from input to output without cycles. Includes MLPs (multi-layer perceptrons) but not RNNs which have loops.

56. What is the key difference between a multi-layer perceptron and a single-layer perceptron?

Correct Answer: a

Explanation: MLPs have one or more hidden layers between input and output, enabling them to learn non-linear decision boundaries and solve XOR-like problems.

57. In a multi-layer perceptron (MLP), which activation function is used in every node?

Correct Answer: c

Explanation: Traditional MLPs used sigmoid in hidden layers (though modern networks often use ReLU). The key is any non-linear activation enables universal approximation.

58. Which type of neural network is specifically used for computer vision tasks such as image recognition?

Correct Answer: c

Explanation: CNNs use convolutional layers to preserve spatial relationships and share parameters, making them highly effective for image data.

59. In a CNN, what does the kernel filter do?

Correct Answer: b

Explanation: Kernels (filters) perform convolution operations to detect features like edges, textures, or patterns at different locations in the input.

60. In a CNN, how does a kernel filter move over the input data?

Correct Answer: a

Explanation: By default, stride=1 means the kernel slides one pixel at a time (horizontally then vertically), producing a detailed feature map.

61. What is the purpose of a stride filter in CNN?

Correct Answer: c

Explanation: Stride > 1 skips pixels, reducing feature map size and computational cost while capturing broader patterns (trade-off between detail and efficiency).

62. The purpose of a Convolutional Neural Network (CNN) is to handle?

Correct Answer: a

Explanation: CNNs excel at data with grid structure: images (2D grids of pixels), time series (1D), or volumetric data (3D). They exploit spatial locality.

63. In the context of CNNs, what does the pooling layer do?

Correct Answer: b

Explanation: Pooling (max/average) reduces spatial dimensions, providing translation invariance and decreasing parameters to prevent overfitting.

64. What is the term for the step where a CNN converts the feature map into a single column?

Correct Answer: a

Explanation: Flattening reshapes the 2D/3D feature maps into a 1D vector to feed into fully connected layers for final classification/regression.

65. Which formula is used to calculate the feature map size when no stride is applied?

Correct Answer: c

Explanation: Input = 7, Kernel = 3
Feature Map Size = (7 - 3) + 1 = 5 ‚Üí 5 * 5

66. Which formula is used to calculate the feature map size when stride is applied?

Correct Answer: c

Explanation: Input = 7, Kernel = 3, Stride = 2
Feature Map Size = (7 - 3)/2 + 1 = 3 ‚Üí 3 * 3

67. What is the purpose of padding filter in CNNs?

Correct Answer: b

Explanation: Padding adds zeros (or other values) around the input to preserve spatial dimensions or ensure border pixels are considered equally.

68. Which of the following is true about Average Pooling in CNN?

Correct Answer: c

Explanation: Average pooling takes the mean of values in the pooling window, producing smoother feature maps than max pooling.

69. Which of the following is true about Max Pooling in CNN?

Correct Answer: c

Explanation: Max pooling retains the most activated feature (largest value) in each window, providing translation invariance and preserving strong features.

70. What is the main feature of a Recurrent Neural Network (RNN)?

Correct Answer: b

Explanation: RNNs have loops allowing information persistence through hidden states, making them suitable for sequential data like time series or text.

71. Which unit in an RNN maintains the hidden state?

Correct Answer: c

Explanation: Recurrent neurons maintain a hidden state that gets updated at each timestep, capturing information from previous sequence elements.

72. Why are LSTM and GRU used in RNNs?

Correct Answer: c

Explanation: LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Unit) use gating mechanisms to mitigate vanishing gradient problem, allowing learning of long-range dependencies.

73. Which of the following is an important feature of a Recurrent Neural Network (RNN)?

Correct Answer: b

Explanation: The hidden state acts as memory, allowing RNNs to capture temporal dynamics in sequences like text, speech, or time-series data.

74. Which of the following are applications of Recurrent Neural Networks (RNNs)?

Correct Answer: d

Explanation: RNNs excel at sequential tasks: NLP (language modeling), speech processing, time series forecasting. CNNs are better for image tasks, but hybrid models exist.

75. What is the main purpose of backpropagation in a neural network?

Correct Answer: b

Explanation: Backpropagation uses chain rule to compute gradients of loss with respect to each parameter, enabling gradient descent optimization.

76. Which optimization algorithms are commonly used with backpropagation?

Correct Answer: b

Explanation: Gradient descent variants (SGD, Adam, RMSprop) use backpropagation gradients to update weights iteratively, minimizing the loss function.

77. What is backpropagation in a neural network?

Correct Answer: c

Explanation: Backpropagation is the core training algorithm for neural networks, propagating error gradients backward through the network to update weights.

78. Probabilistic modeling is mainly used to:

Correct Answer: b

Explanation: Probabilistic models (Bayesian networks, Gaussian processes) provide uncertainty estimates along with predictions, crucial for risk-sensitive applications.

79. Which term describes: Model predicts positive, and it's actually positive?

Correct Answer: c

Explanation: True Positive: correct prediction of the positive class. Example: correctly identifying a disease in a patient who has it.

80. Which term describes: Model predicts positive, but it's actually negative?

Correct Answer: b

Explanation: False Positive: Type I error. Example: diagnosing a healthy person as sick (false alarm).

81. If there are many True Negatives (TN), it means the model predicts correctly.

Correct Answer: a

Explanation: True Negatives are correct predictions of the negative class. High TN indicates good performance on negative instances.

82. What is the key characteristic of nonlinear regression?

Correct Answer: b

Explanation: Nonlinear regression models relationships that can't be expressed as straight lines, using curves (polynomial, exponential, logarithmic, etc.).

83. Which of the following is an example of nonlinear regression?

Correct Answer: b

Explanation: Polynomial equations (like quadratic) are nonlinear in the parameters relative to the input variables, producing curved relationships.

84. What is the main difference between linear and nonlinear regression?

Correct Answer: b

Explanation: Linear: y = Œ≤‚ÇÄ + Œ≤‚ÇÅx (straight line). Nonlinear: any functional form not linear in parameters (curves, more complex shapes).

85. What does "overfitting" refer to in the context of regression models?

Correct Answer: c

Explanation: Overfitting occurs when a model learns training noise/details too well, losing generalization ability. High variance, low bias.

86. What does "underfitting" refer to in the context of regression models?

Correct Answer: b

Explanation: Underfitting: model is too simple to capture underlying patterns. High bias, low variance. Both training and test performance are poor.

87. What is the main advantage of using Polynomial Regression?

Correct Answer: b

Explanation: Polynomial regression can model curved relationships by adding polynomial terms (x¬≤, x¬≥, etc.) while remaining linear in parameters for estimation.

88. Which combination usually indicates overfitting?

Correct Answer: c

Explanation: Low bias (fits training well) + high variance (sensitive to fluctuations) = overfitting. The model is too complex relative to data.

89. Increasing the degree of a polynomial in regression may cause:

Correct Answer: b

Explanation: Higher polynomial degrees increase model complexity, potentially fitting noise in training data and reducing generalization (overfitting).

90. What is the target when choosing a polynomial degree?

Correct Answer: b

Explanation: Goal: find simplest model (lowest degree) that captures essential patterns (bias-variance tradeoff). Use validation/cross-validation to select optimal degree.
